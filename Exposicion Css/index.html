Metodologías CSS: Atomic Design & SMACSS

Atomic Design

Definición: Atomic Design es una metodología de diseño basada en la construcción de interfaces usando componentes modulares y reutilizables, inspirados en la química atómica.

Características:

  • Componentes divididos en átomos, moléculas, organismos, plantillas y páginas.
  • Reutilización de componentes para escalar proyectos grandes.
  • Facilidad para mantener el código.

Usos:

  • Sistemas de diseño modular.
  • Interfaces escalables y mantenibles.

Ejemplos:

Como formulario, un sistema de diseño basado en Atomic Design puede incluir componentes como:

  • Átomo: Un botón de "Enviar".
  • Molécula: Un campo de texto junto con una etiqueta.
  • Organismo: Un formulario de registro que combina varios campos de entrada y botones.

Frameworks como React y Vue permiten construir interfaces usando componentes que se pueden reutilizar en diferentes partes de la aplicación, facilitando la escalabilidad y el mantenimiento, siendo estos librerías.

SMACSS

Definición: SMACSS (Scalable and Modular Architecture for CSS) es una metodología para estructurar y organizar hojas de estilo CSS de manera modular y escalable.

Características:

  • Organización de CSS en cinco categorías: Base, Layout, Module, State, y Theme.
  • Modularidad y reusabilidad de componentes CSS.
  • Escalabilidad en proyectos grandes.

Usos:

  • Proyectos de gran escala con múltiples desarrolladores.
  • Estilos modulares y mantenibles.

Ejemplos:

Una estructura de carpetas utilizando SMACSS podría ser:

  • Base: Reset y estilos generales.
  • Layout: Estilos de la estructura general de la página.
  • Module: Estilos específicos de componentes como tarjetas o botones.
  • State: Estilos para estados como "activo" o "inactivo".
  • Theme: Colores y tipografía que se aplican en toda la aplicación.

En un sitio web de comercio electrónico, un módulo de tarjeta de producto puede incluir una imagen, un título, una descripción y un botón de "Agregar al carrito", permitiendo su reutilización en diferentes páginas de productos, siendo este un claro ejemplo de los Módulos reutilizables
